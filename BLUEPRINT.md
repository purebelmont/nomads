## 4. 서비스 상세

### 4.1 드론 촬영 (Drone Filming)
- **설명:** 최첨단 드론 장비로 대형 건설 현장, 조선소, 제조 시설의 스케일을 압도적인 영상으로 담아냅니다.
- **포함 항목:**
  - 산업용 고성능 드론 촬영
  - 4K/6K 항공 영상
  - 대형 시설 조감도 촬영
  - 공정 진행 모니터링 영상
  - 항공 파노라마/타임랩스

### 4.2 기업 홍보영상 (Corporate Video)
- **설명:** 기업의 비전과 가치를 시네마틱 영상으로 전달합니다. 브랜드 스토리텔링의 새로운 기준.
- **포함 항목:**
  - 기업 브랜드 필름
  - CEO/임원 인터뷰 영상
  - 시설 투어 영상
  - IR/투자 유치용 영상
  - 채용 홍보 영상

### 4.3 제품 촬영 (Product Shoot)
- **설명:** 제품의 디테일과 품질을 극대화하는 전문 촬영. 산업 장비부터 완성품까지.
- **포함 항목:**
  - 산업 장비/중장비 촬영
  - 제품 디테일 매크로 촬영
  - 360도 회전 촬영
  - 카탈로그/웹사이트용 이미지
  - 제품 시연 영상

### 4.4 광고 영상 (Commercial Production)
- **설명:** 임팩트 있는 광고 영상 제작. 기획부터 후반 작업까지 올인원 프로덕션.
- **포함 항목:**
  - TV/디지털 광고 제작
  - SNS 숏폼 콘텐츠
  - 전시회/박람회용 영상
  - 모션그래픽/CG 합성
  - 나레이션/BGM 제작

## 12. Supabase 데이터베이스

### quote_requests 테이블

```sql
CREATE TABLE quote_requests (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  created_at TIMESTAMPTZ DEFAULT now() NOT NULL,
  company_name TEXT NOT NULL,
  contact_name TEXT NOT NULL,
  email TEXT NOT NULL,
  phone TEXT NOT NULL,
  service_type TEXT NOT NULL,
  budget_range TEXT,
  timeline TEXT,
  project_description TEXT NOT NULL,
  is_read BOOLEAN DEFAULT false,
  status TEXT DEFAULT 'new'
);

ALTER TABLE quote_requests ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Allow anonymous insert" ON quote_requests
  FOR INSERT TO anon WITH CHECK (true);
```
